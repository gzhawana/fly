# A股 Trading Monitoring System

## Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/gzhawana/fly.git
   cd fly
   ```
2. **Install dependencies**:
   ```bash
   npm install  # or pip install -r requirements.txt
   ```

## Usage
1. **Run the application**:
   ```bash
   npm start  # or python app.py
   ```
2. **Access the monitoring dashboard** at `http://localhost:3000`.

## Features
- Real-time A股 trading monitoring
- Notifications for significant price changes
- Historical data analysis
- User-friendly interface

## Architecture
- **Frontend**: Built with React/Angular
- **Backend**: Node.js/Python Flask
- **Database**: MongoDB/MySQL
- **APIs**: Provides data through RESTful APIs

For more details, please refer to the respective documentation of each component.
